deCODE Hackathon with KOHO

Overview – Donation feature on KOHO mobile app

Role – Designer along with 2 other designers and 8 developers

Tools – Figma, Miro

Context

In March 2021, I attended the deCODE hackathon and was paired with KOHO, along with five other students. We were given less than two days to design and develop a fully functioning feature that would integrate with KOHO’s existing platform. It was a lot to take on, but thankfully there were three full timers from KOHO (one designer and two developers) to mentor us!

KOHO creates financial products that are open, intuitive, and designed to help users live a better life. One of their key features is something called RoundUps, where every transaction is rounded up to the nearest $1, $5, or $10, and the extra money is added to a pool. Overtime, this pool accumulates more and more and you can use it as savings money. See more about this cool feature here.

KOHO approached us with the idea to implement some sort of donation feature into their mobile app. There were many routes that we could explore in regards to donations, but we wanted to choose a path that was feasible in the limited timeframe.

Goal

To implement a donation feature into the existing KOHO mobile application that would allow users to search for charities, set donation goals, and continuously add to these goals from their RoundUps.

Target Users

Due to the short timeframe, we were unable to conduct proper user research and testing. Instead, we tried to gather as many opinions and thoughts from the team.

The target audience for this feature would be existing KOHO users.

Next Steps

If I were to take this project even further, I would implement the following features:

  • Allowing user to set more than one donation goal at a time
  • Providing the option to start a social campaign where groups of friends can contribute to the same donation goal
  • Allowing the user to immediately donate if they have enough money in their RoundUps pool
  • Filtering charities by location, interest, popularity, type, etc.

Conclusions

deCODE was my very first hackathon and partnering with KOHO made it even more special. Being able to work alongside KOHO was a great opportunity and I was able to learn and gather so much valuable feedback.

Overall, I am very thankful for this experience and was able to meet some new people. The design iterations and mindset that we had can be seen in our final product. I can definitely see myself bringing what I’ve learned into my future career.
